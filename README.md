-# Программа создания расписания аничкого лицея
1) В файле с алгоритмом (без интерфейса) уже введены все данные по моей школе. Данные о нагрузке и о доплнительных уроках в школе прилагаются в этой ветке
2) Исключения самих учителей выписаны из приватной таблицы, доступ к которой невозможен
3) Если вы используете программу только с алгоритмом, то txt файл с окнами и excel файл с распианием будут созданы в дирекотории с самой программой
4) Чтобы изменять количетство классов в расписании, стоит просто комментировать их (отосится к программе только с алгоритмом)
5) На строчке 2467 после знака "=" можно выбрать количество секунд для перебора перебора вариантов (отосится опять же только к программе с алгоритмом)
